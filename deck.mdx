## Inicializan el proyecto localmente

- git init
- git add .
- git commit -m "first commit"
- git branch -M main
- git remote add origin 'remote url'
- git push -u origin main

## Handy commands

import diff from './assets/diff.png';

- git add . // Agregar todos los cambios.
- git commit -m "" // Dar una descripción de esos cambios
- git status // Saber que archivos haz cambiado, agregado o eliminado.
- git log // Historial de commits 
- git diff // Diferencia entre lo que existia en tu comentario previo y tus cambios actuales.
- git push // Subir cambios a rama remota.
